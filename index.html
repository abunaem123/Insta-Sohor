At first the Web page will be loaded in Browser, then it will parse HTML and create DOM, which will show a live view of the page. After that, the browser will take all pages linked to the HTML file. CSS file will be one of them, which information comes from CSS parser.

Then DOM will put the HTML and CSS file together and a web page will be created and then JS file will start loading but after the HTML and CSS file finish loading. JS file will be loaded by Javascript engine load.Once everything is done, the JS file starts executing according to the order in the JS file. The order in the file is important.

LocalStorage: Type of web storage object availabne in html 5 that allows storing data with no expiration date. Designed for storage that spans multiple windows and it lasts beyond the current session. It's not session based, need to be deleted via JS or manually client side reading only and less older browsers support.
SessionStorage: Type of web storage object available in html 5 that allows storing data for one session. Allows the user to carry out a single transaction but could be carrying out multiple transaction in defferent windows at the same time.It's session based and working per window or tab client side reading only less older browsers support.
